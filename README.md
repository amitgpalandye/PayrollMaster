# 🏢 PayrollMaster - Enterprise Payroll Management System

A comprehensive, feature-rich Payroll Management System built with Flask, MySQL, and modern web technologies. Streamline employee management, salary processing, attendance tracking, and leave management with an intuitive interface.

## ✨ Features

### Core Functionality
- **Employee Management** - Add, edit, delete, and view employees with department categorization
- **Department Organization** - Automatically group and organize employees by department
- **Salary Processing** - Calculate net salary with allowances and deductions
- **Payslip Generation** - Generate detailed payslips using stored procedures
- **Attendance Tracking** - Record and monitor employee attendance by month
- **Leave Management** - Submit leave requests with automatic 20-day annual limit enforcement
- **Bonus Awards** - Award and track employee bonuses
- **Real-time Dashboard** - View key metrics including total employees, departments, payroll, and average salary

### Technical Features
- **10 SQL Analytical Queries** - Complex queries including JOINs, GROUP BY, HAVING, Subqueries, and Correlated Subqueries
- **Stored Procedures** - Automated salary calculation and payslip generation
- **Database Triggers** - Automatic salary adjustment based on attendance, leave limit enforcement
- **Modern Responsive UI** - Built with Tailwind CSS with smooth animations and dark theme
- **RESTful API** - Complete CRUD operations via Flask backend
